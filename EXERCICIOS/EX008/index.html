Principais formatações.

Negrito/destaque

Nessa frase temos uma frase em negrito usando a tag <b> (não-semântica, só forma)

Nesta frase, temos um termo em destaque usando a tag <strong>, que é semântica e significa que o que está entre a tag é um termo destacado.

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não-semântica)

Nesta frase, temos um termo em itálico usando a tag <em>, que é semântica.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno. A tag <small> não é obsoleta.

Texto deletado

Podemos marcar um texto excluido para indicar que ele não será mais considerado.
